SynonymReplacer(props)

SynonymReplacer component for replacing a selected word with its synonym.
Parameters:
Name Type Description
props Object The props for SynonymReplacer.
Properties
Name Type Description
active boolean Whether the dialog box is active.
onClose function Function to execute on close of the dialog box.
selectedWord Object The currently selected word and its details.
Properties
Name Type Description
index number The index of the selected word.
word string The selected word.
colour string The colour of the selected word.
synonyms Array The list of synonyms for the selected word.
Source:

Example
const selectedWord = {
  index: 1,
  word: 'example',
  colour: 'blue',
  synonyms: [
    { synonym: 'sample', rank: 1, lemma: 'sample' },
    { synonym: 'instance', rank: 2, lemma: 'instance' },
  ]
};
return (
  <SynonymReplacer active={true} onClose={() => {}} selectedWord={selectedWord} />
)
